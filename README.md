# POEP2
# Cybersecurity Awareness Bot — Part 2

A C# WPF chatbot that educates users about cybersecurity threats and how to stay safe online through an interactive graphical interface.

# How to Run

Open the project in Visual Studio  
Press F5 to run  
Enter your name when prompted  
Type `menu` to see all available topics  
Type a number (1–6) to get cybersecurity advice  
Type `back` to return to the menu  
Type `exit` to quit  

# Features

Voice greeting using `.wav` audio  
WPF graphical user interface  
Personalised user interaction  
Name input popup  
Menu-driven chatbot system  
Delegate and event usage  
Numbered cybersecurity topics  
Interactive chatbot responses  
Navigation commands (`menu`, `back`, `exit`)  
GitHub Actions CI workflow  

# Cybersecurity Topics

1. Password Safety  
2. Phishing  
3. Malware  
4. Privacy  
5. Wi-Fi Security  
6. Two-Factor Authentication (2FA)  

# Project Structure
